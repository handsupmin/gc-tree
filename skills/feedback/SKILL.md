# gctree Feedback

Use this when the user wants to adjust a proposed global-context update before it is applied.

## Procedure
1. Read the pending proposal.
2. Capture the user's requested changes.
3. Revise the proposal summary and affected docs.
4. Re-show the revised summary.
5. Keep the proposal in the proposal stage until the user explicitly approves it.
